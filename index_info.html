Info

Spring Clicker is an MIT open-source* automation software powered with selenium framework. This project is meant to get financial support for restoring Democracy, Freedom and Justice and Human Rights in Burma ( Myanmar ).

The original websites used in the software will be managed and maintained by their owner.

For suggestions, making reviews and reporting bugs or user experiences, you can check out the Twitter page or Git Hub.

* Open-Source means it’s free to use and can be studied or improved by the public.
